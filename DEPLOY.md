# Deployment

1. Open the public repository `LiRongji1995/LiRongji1995`.
2. Upload `README.md`, the `assets` directory, and the `.github` directory to the repository root.
3. Commit the changes.
4. Open the repository's **Actions** tab.
5. Run **Generate contribution snake** once manually.
6. After the workflow creates the `output` branch, the contribution snake will appear in the README.

The large dashboard artwork is stored locally at `assets/eva-profile-dashboard.png`, so it does not depend on an external image host.
